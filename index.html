How to get a Client ID
Gmail (Google Cloud)
  1. Go to console.cloud.google.com
  2. Create a new project (any name)
  3. Go to APIs & Services → Library → enable Gmail API
  4. Go to APIs & Services → Credentials → Create OAuth client ID
  5. Type: Web application
  6. Authorized JavaScript origins: add your GitHub Pages URL
  7. Copy the Client ID and paste it in the app
Outlook (Azure Portal)
  1. Go to portal.azure.com → App Registrations → New
  2. Name it, select "Accounts in any organizational directory"
  3. Copy the Application (client) ID
  4. Go to Authentication → Add platform → Single-page application
  5. Redirect URI: your GitHub Pages URL
  6. Go to API permissions → Add: Mail.Read and User.Read
